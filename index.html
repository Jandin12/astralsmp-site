About the server

Welcome to Astral SMP

A survival multiplayer server built for a friendly, yet very competitive community. Whether you want to build, grind, or PvP, there's a place for you.

Survival, Semi-Vanilla

Classic survival with a twist: each season gets a new plugin that completely changes the style of gameplay. From PowerGems to Infuse SMP, the server dynamically changes.

Friendly Community

An active, welcoming player base and staff team who actually play on the server.

Seasonal Events

Regular community builds, mini-games, and events to keep every season interesting.

Griefing Protected

Land claims keep your builds safe, so you can focus on creating, not defending.

Getting started

How to Join

Up and running in under a minute — on Java or Bedrock.

  1. 1

    Pick your platform

    Astral SMP runs GeyserMC, so both Minecraft: Java Edition and Bedrock Edition (mobile, console, Windows) can join and play together.

  2. 2

    Add the server

    Java: play.astralsmp.org (default port 25565). Bedrock: play.astralsmp.org, port 19132.

  3. 3

    Register & log in

    The server is cracked — no premium account needed — and secured with an advanced password plugin. Register a password on your first join, then log in each visit after.
